# dsa-java roadmap

Curriculum order for Phases 1–2. Tick status in [`../../PROGRESS.md`](../../PROGRESS.md).

## Phase 1 — data structures (reference, in build order)
1. **linear/** — DynamicArray → SinglyLinkedList → DoublyLinkedList → Stack → Queue → Deque → RingBuffer
2. **hashing/** — HashMapChaining → HashMapOpenAddressing → HashSet → LRUCache
3. **trees/** — BinaryTree → BinarySearchTree → AVLTree → BinaryHeap → Trie → SegmentTree → FenwickTree
4. **graphs/** — Graph → UnionFind
5. **algorithms/sorting/** — MergeSort → QuickSort → HeapSort → CountingSort
6. **algorithms/searching/** — BinarySearch (+ lower/upper bound, rotated)
7. **algorithms/graph/** — GraphTraversal (BFS/DFS) → TopologicalSort → Dijkstra → MinimumSpanningTree

## Phase 2 — coding (NeetCode 150, the 18 patterns)
Arrays & Hashing · Two Pointers · Sliding Window · Stack · Binary Search · Linked List · Trees ·
Heap/PQ · Backtracking · Tries · Graphs · Advanced Graphs · 1-D DP · 2-D DP · Greedy · Intervals ·
Math & Geometry · Bit Manipulation.

**Arrays & Hashing (9), first set:** Contains Duplicate (E) · Valid Anagram (E) · Two Sum (E) ·
Group Anagrams (M) · Top K Frequent Elements (M) · Encode/Decode Strings (M) · Product of Array
Except Self (M) · Valid Sudoku (M) · Longest Consecutive Sequence (M).
