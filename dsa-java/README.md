# dsa-java

Lean Java project for **Phase 1 (data structures)** and **Phase 2 (coding / NeetCode 150)** of
[`../STUDY-PLAN.md`](../STUDY-PLAN.md). No framework — just Java 21 + JUnit 5, for a fast loop.

- **Package:** `com.venkat.dsa`
- **Reference impls** (`linear/ hashing/ trees/ graphs/ algorithms/`) — fully documented; read to study.
- **Coding practice** (`coding/{easy,medium,hard}/`) — NeetCode 150 stubs you solve until tests pass.
- **Docs:** conventions, roadmap, and templates live in [`../docs/dsa/`](../docs/dsa/).

## Run

```powershell
.\mvnw.cmd test
```

Start with `src/main/java/com/venkat/dsa/linear/DynamicArray.java` (see the study plan).
